## 📂 專案列表與功能說明

### 1. WAV 音效檔播放器 (上課練習 3)
本專案實作了一個專門播放 `.wav` 格式的音訊播放器，重點在於資源管理與播放控制。
* **核心技術**：`System.Media.SoundPlayer` 類別、`System.IO.FileStream`。
* **主要功能**：
    * **瀏覽檔案**：使用 `OpenFileDialog` 過濾並選取音訊。
    * **播放模式**：支援單次播放與循環播放 (`PlayLooping`)。
    * **資源釋放**：實作停止播放功能並妥善關閉檔案串流。
    * **防誤觸機制**：表單關閉前會跳出 MessageBox 確認視窗。

### 2. 簡易電子琴 (上課練習 4)
透過呼叫 Windows 系統底層 API，模擬電子琴發聲並實作響應式 UI 佈局。
* **核心技術**：Windows API `Beep` 函式 (`kernel32.dll`)、`Dictionary` 記錄 UI 初始狀態。
* **主要功能**：
    * **頻率發聲**：根據 8 個按鈕對應的音階頻率 (Do-Do) 調用系統嗶聲。
    * **事件共享**：多個按鈕共用同一個事件處理函式，提高程式碼重用性。
    * **動態縮放**：實作 `SizeChanged` 事件，讓琴鍵隨視窗大小變動自動等比例縮放。

### 3. 多媒體播放器 (上課練習 5)
整合第三方 COM 元件實作功能完整的影音播放程式。
* **核心技術**：Windows Media Player COM 元件、`AxWMPLib`。
* **主要功能**：
    * **影音相容**：支援 `.wmv`、`.mp4`、`.avi` 等多種影片格式。
    * **自定義介面**：隱藏原生控制列 (`uiMode = "none"`)，改用自製按鈕控制播放、暫停與停止。
    * **檔案串接**：動態載入選取路徑並即時更新播放狀態。
